# SakMvp1 핵심 기능 목록

## 사용자 인증 및 계정 관리
- 이메일/비밀번호 기반 로그인
- 회원가입 (Firebase Authentication)
- 자동 로그인 설정
- 관리자/일반 사용자 권한 분리
- 로그아웃 기능

## 졸업 요건 분석
- 학번/학부/트랙 선택
- 추가 졸업 요건 선택 (복수전공, 부전공, 교직 등)
- 수강 과목 입력 (카테고리별)
  - 전공필수/전공선택/전공심화/학부공통
  - 교양필수/교양선택
- 직접 입력 과목 추가 기능
- 졸업 요건 충족 여부 자동 계산
- 카테고리별 학점 집계
- 부족/초과 학점 분석
- 대체과목 규칙 자동 적용
- 넘치는 학점 자동 이월 처리
- 도넛 차트 시각화
- 분석 결과 PDF 내보내기
- 분석 결과 저장 및 불러오기

## 과목 추천
- 학년/학기별 추천 과목 제공
- 우선순위 기반 추천 알고리즘
  - 부족한 카테고리 우선
  - 남은 학점이 많은 카테고리 우선
  - 사용자 지정 카테고리
- 필수 과목 강조
- 선이수 과목 확인
- 추천 과목 상세 정보 표시

## 시간표 관리
- 주간 시간표 보기
- 월간 시간표 보기
- 시간표 일정 추가/수정/삭제
- 여러 시간표 저장 및 전환
- 시간표 그룹 관리
- 공용 캘린더 연동

## 캠퍼스 지도
- OpenStreetMap 기반 캠퍼스 지도
- 건물별 위치 마커
- 시설별 필터링
  - 복사기 위치
  - 프린터 위치
  - ATM 위치
  - English Cafe 위치
- 지도 스크롤 영역 제한 (캠퍼스 경계)
- 줌 레벨 제한
- 현재 위치 표시
- 나침반 오버레이

## 홈 화면
- 배너 슬라이더
- 바로가기 메뉴
  - 졸업 요건 분석
  - 과목 추천
  - 필요 서류
  - 자격증 정보
  - 챗봇
  - 급식 메뉴
- 공지사항 표시

## 내정보
- 사용자 프로필 관리
- 학사 정보 설정 (학번, 학부, 트랙)
- 저장된 분석 결과 조회
- 북마크한 자격증 목록
- 설정
  - 색약 모드 (흑백 필터)
  - 접근성 옵션
- 로그아웃

## 자격증 정보
- 자격증 목록 조회
- 카테고리별 필터링
  - 국가자격증
  - 민간자격증
  - 국제자격증
- 자격증 상세 정보
- 북마크 기능
- 관련 과목 정보

## 필요 서류
- 학과별 필수 서류 안내
- 교양 관련 서류
- 폴더별 분류
- 서류 파일 보기
- 이미지 확대/축소 기능

## 체크리스트
- 할 일 목록 관리
- 항목 추가/수정/삭제
- 완료 여부 체크
- Firebase 동기화

## 식당 메뉴
- 당일 급식 메뉴 조회
- 웹 크롤링 기반 실시간 정보

## 챗봇
- 학사 정보 자동 응답
- 미리 정의된 FAQ

## 관리자 기능
- 졸업 요건 관리
  - 학번/학부/트랙별 요건 추가/수정/삭제
  - 카테고리별 과목 관리
  - 학점 요구사항 설정
  - 대체과목 규칙 관리
- 학생 데이터 조회
  - 학생 목록 보기
  - 개별 학생 상세 정보
  - 수강 이력 확인
- 배너 관리
  - 배너 추가/수정/삭제
  - 이미지 업로드
  - 표시 순서 관리
  - 학부별 표시 설정
  - 활성화/비활성화
- 자격증 관리
  - 자격증 정보 추가/수정/삭제
  - 카테고리 분류
  - 관련 과목 연결
- 서류 관리
  - 학과별 서류 폴더 관리
  - 서류 파일 업로드
  - 일반 서류/학과 서류 분리 관리

## 공통 기능
- Firebase 실시간 동기화
- 오프라인 캐싱 (메모리 캐시)
- 로딩 상태 표시
- 에러 처리 및 사용자 피드백
- 뒤로가기 네비게이션
- Material Design UI
- 다크/라이트 테마 (색약 모드)
- EdgeToEdge 디스플레이 지원
- 시스템 UI 인셋 처리

## 데이터 관리
- Firebase Firestore 연동
- 로컬 SharedPreferences 저장
- 이미지/파일 Firebase Storage 저장
- 5분 TTL 메모리 캐시
- N+1 쿼리 최적화
- 대체과목 자동 매핑
- 학점 이월 자동 처리
