## 1. 정확한 기술 내용 (수정 불필요)

### ✅ 프로젝트 코드와 일치하는 내용들

다음 기술 내용들은 실제 프로젝트 코드를 분석한 결과 **정확하게 기재**되어 있습니다:

#### 1-1. 시스템 아키텍처 (Page 1)
- ✓ **Firebase 서버리스 아키텍처** 채택
- ✓ **Firebase Authentication** 사용자 인증
- ✓ **Cloud Firestore** 및 **Realtime Database** 데이터 저장
- ✓ **Firebase Storage** 파일 관리

#### 1-2. 개발 환경 (Page 1, 4)
- ✓ **Java 11 기반** 개발
  - 실제 코드: `sourceCompatibility = JavaVersion.VERSION_11`
  - 실제 코드: `targetCompatibility = JavaVersion.VERSION_11`

- ✓ **Android API 26~35 지원**
  - 실제 코드: `minSdk = 26`
  - 실제 코드: `compileSdk = 35`
  - 실제 코드: `targetSdk = 35`

#### 1-3. 사용 라이브러리 (Page 1)
- ✓ **Material Design 컴포넌트** (Material 1.13.0)
- ✓ **Glide 4.16.0** - 이미지 로딩 및 캐싱
- ✓ **Jsoup 1.17.2** - HTML 파싱 (급식 메뉴 스크래핑)
- ✓ **OkHttp 4.12.0** - HTTP 네트워크 통신
- ✓ **iText 7.2.5** - PDF 문서 생성
- ✓ **PhotoView 2.3.0** - 이미지 확대/축소
- ✓ **OSMDroid 6.1.18** - 지도 기능

#### 1-4. 핵심 알고리즘 (Page 4-6)
- ✓ **대체과목 자동 매칭 알고리즘** 구현 확인
  - `ReplacementRule` 모델 클래스 존재
  - `ReplacementRulesFragment` 구현 확인
  - 조건부 대체 규칙 지원 (학번별, 학기별)

- ✓ **학점 이월 처리 알고리즘** 구현 확인
  - 우선순위 기반 연쇄 이월 처리
  - `calculateCategoryCreditsWithOverflow` 메서드 존재

- ✓ **과목 추천 알고리즘** 구현 확인
  - `CourseRecommendationActivity` 존재
  - 다중 가중치 기반 추천 시스템

- ✓ **웹 스크래핑 (급식 메뉴)** 구현 확인
  - `MealMenuActivity`에서 Jsoup 사용
  - 패턴 기반 게시글 필터링

#### 1-5. PDF 생성 기능 (Page 7-8)
- ✓ **iText 7 라이브러리** 사용
- ✓ **한글 폰트 임베딩** (NanumGothic.ttf)
- ✓ **PDF 내보내기 기능** 구현
